# Table

## *How to create a table*

Tables in Markdown are created using **pipe (|) and dash (-) symbols**. 
***
**For Example:**
***

|S.no | Fruits | Vitamins |
| --- | ---    |  ---     |
| 1.  |  Apple |    C     |
| 2.  | Banana |    B     |
| 3.  | Papaya |    A     |
| 4.  | Orange |    C     |
